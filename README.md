#  Clinic Appointment Tracker

A simple clinic management app built using Python and Streamlit.

##  Features
- Add patient details
- Auto-generate Customer ID
- Search using Customer ID or Mobile Number
- Track status (Active / Completed)
- View Active Patients Dashboard

##  Tech Stack
- Python
- Streamlit
- Pandas

##  How to Run
1. Install dependencies:
   pip install streamlit pandas

2. Run the app:
   streamlit run app.py

3. Open in browser:
   http://localhost:8501

##  Files
- app.py → Main application
- patients.csv → Stores data (auto-created)

##  Purpose
Built to simulate a simple real-world clinic workflow and practice building data-driven applications.
